These projects are things that I have done in my college.

Recreating an Amazon page

In this project we had to recreate an Amazon page. To do this I had to create a h1 tag and then create a paragraph under it with the links. This was the easy part, because creating those tags are as simple as writing the information down. The images were the hard part, because I needed to get them all to be aligned with a certain number of images on each line. I solved this by getting each image to be a certain width and height, but there are multiple other ways that this could have been solved.

Recreating a Github page

This Project required us to recreate a Guthub page. This was a challenge because of the images at the bottom. The top part was fairly straight forward though. It was just typing in the text and then getting the text to appear in the middle of the screen. The bottom required a little bit more though. It required you put the image and paragraph in a section and then float it to the left to get both of them on the same line. The two sections also needed to be a width of 50% in order to get them both on the same line.
